# @spartan-ng/brain/time-input

A headless, accessible time input primitive for 12-hour format time selection.

## Features

- 12-hour format with AM/PM period
- Keyboard navigation (Arrow keys to increment/decrement, direct digit entry)
- Segment-based editing (hours, minutes, period)
- `ControlValueAccessor` support for Angular forms
- Fully accessible with ARIA spinbutton roles
